# 有关游戏的杂文都在这里

个人心中，打游戏不是一个有益的爱好。和体育运动相比，它不能锻炼你的体魄，无法更让你大方外向，反而会让你宅在屋子里，久坐伤腰，顺带让你大拇指患上腱鞘炎。和音乐、美术相比，不能让你与朋友去唱歌时显露风头，不能在各种艺术展中出众。

其实以上都是说的都是怪话，希望大家能在这里抓住第九艺术的尾巴。应该把游玩一款游戏和读一本书来类比，无论是箱庭还是开放世界，不论各类游戏种类各类机制，玩一款优秀的游戏与读完一本书是一样的，镜头、光影、音乐、剧本还有最重要的核心机制，总有能打动玩家的地方。不过一款游戏的表现是这些的集合，一款好的游戏需要控制好项目体量，突出核心体验，平衡好各个要素。
